# MM_trace

# Using CHESSCC or PEANO?
Change the value of ENABLE_CHESSCC in Makefile.

# Kernel

```
make kernel
```

It will generate mm.o.

# Bitstream

```
make bitstream
```

It will generate xclbin file.

# Instruction
```
make instructions
```

It will generate txt file.

# Host

```
make host
```

It will generate run.exe.

# Trace
```
make trace
```

It will generate json file.
